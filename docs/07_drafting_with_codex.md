# 07 用 Codex 起草综述正文

## 推荐顺序

1. 先写大纲；
2. 每一节先列 claims；
3. 每个 claim 绑定 Zotero 文献；
4. 再写段落；
5. 最后统一润色。

## 段落写作模板

```text
请基于下面 evidence matrix 和章节目标，写一个综述段落。
要求：
1. 每个关键论断后标注 [CitationKey]；
2. 不要新增文献；
3. 不要过度确定，保留争议；
4. 输出中文学术风格。
```

## 防止 AI 幻觉

- 不让 AI 直接输出完整参考文献；
- 不接受没有 CitationKey 的事实性论断；
- 所有引用回到 Zotero 核对；
- Word 成稿阶段用 Zotero 插件插入正式引用。
