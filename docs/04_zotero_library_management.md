# 04 Zotero 文献库管理

## Collection 推荐结构

```text
Review_Project_Name/
  00_candidates/
  01_core_classics/
  02_high_impact/
  03_recent_frontier/
  04_methods/
  05_background/
  06_excluded/
  99_used_in_manuscript/
```

## 每篇文献最低元数据要求

- Title
- Author
- Year
- Journal/Conference/Book
- DOI 或 URL
- Abstract
- PDF 附件，如可获得

## Zotero 笔记模板

```text
一句话总结：
研究问题：
方法/设计：
样本/数据：
核心结果：
局限：
可引用观点：
适合放入综述的章节：
```

## 导出建议

阶段性导出 BibTeX/RIS/CSV 到 `zotero_exports/`，用于备份、制表和审计。
