# 02 确定综述主题与写作协议

## 你要先回答 6 个问题

1. 综述领域是什么？
2. 目标读者是谁？临床、工程、基础研究还是跨学科？
3. 是 narrative review、scoping review，还是 systematic review？
4. 时间范围是什么？例如 2015–2026。
5. 主要数据库是什么？PubMed、Web of Science、Scopus、IEEE Xplore、ACM、arXiv 等。
6. 最终投稿目标是什么？中文期刊、英文综述、课程论文、基金背景？

## 推荐产物

填写：`templates/review_protocol.md`

## Codex 提示词

```text
我准备写一篇关于【主题】的综述。请帮我把主题拆成：
1. 核心科学问题；
2. 关键概念和同义词；
3. 可能的章节结构；
4. 检索关键词；
5. 应该重点比较的技术路线/理论流派；
6. 可能的综述表格设计。
要求：不要编造文献，只输出检索和写作框架。
```
