# 03 文献检索：经典、高分、前沿

## 文献池分三层

1. **经典文献**：奠基方法、被频繁引用、定义概念的论文。
2. **高影响文献**：高被引、顶刊/顶会、指南/共识。
3. **前沿文献**：近 2–3 年的新方法、新应用、新争议。

## 推荐检索流程

1. 先用综述/指南找到关键词和核心作者；
2. 在数据库中做系统检索；
3. 用引用追踪补充经典文献；
4. 用近年筛选补充前沿文献；
5. 把候选文献保存到 Zotero 的 `00_candidates` collection。

## 文献筛选标签

- `include`
- `exclude_irrelevant`
- `classic`
- `high_impact`
- `method`
- `clinical`
- `review`
- `meta_analysis`
- `to_read`
- `read_done`

## Codex 提示词

```text
下面是我从数据库导出的文献列表，请帮我按主题、方法、研究对象、年份、期刊影响力进行初筛。
输出：
1. 必读核心文献；
2. 可作为背景的文献；
3. 可能剔除的文献及原因；
4. 需要补检的关键词。
不要新增不存在的文献。
```
