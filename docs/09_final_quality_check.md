# 09 最终质控清单

## 内容质量

- [ ] 主题边界清楚；
- [ ] 每节有明确问题；
- [ ] 经典、高影响、前沿文献均覆盖；
- [ ] 不只是罗列文献，而是比较、归纳、批判；
- [ ] 局限和未来方向具体。

## 引用质量

- [ ] 每个关键论断有来源；
- [ ] Zotero 条目元数据完整；
- [ ] Word 引用是真 Zotero 字段；
- [ ] Bibliography 可刷新；
- [ ] 文内引用与参考文献列表一致。

## 表图质量

- [ ] 每个表图都有明确目的；
- [ ] 表格不重复正文；
- [ ] 图能表达综述框架或证据结构；
- [ ] 表图标题和注释完整。

## 交付文件

- [ ] `final_review.docx`
- [ ] `final_review.pdf`
- [ ] `zotero_export.bib` 或 `.ris`
- [ ] `evidence_matrix.csv/xlsx`
- [ ] `figures/`
- [ ] `quality_check_report.md`
