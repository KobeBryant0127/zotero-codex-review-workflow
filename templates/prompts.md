# Codex 提示词库

## 主题拆解

```text
我想写一篇关于【主题】的综述。请帮我拆解关键词、核心问题、经典文献线索、章节结构和表图设计。不要编造具体引用。
```

## 检索式优化

```text
请根据我的综述问题和纳入标准，生成适用于 PubMed/Web of Science/Google Scholar 的检索式。输出每个检索式的逻辑解释。
```

## 单篇论文提取

```text
请只基于下面论文内容，提取：研究问题、方法、数据/样本、主要发现、局限、可引用观点、适合放入综述哪一节。不确定写 unclear。
```

## Evidence matrix 汇总

```text
下面是 evidence matrix，请帮我归纳主题簇、冲突发现、共识、研究空白和可视化建议。不要新增文献。
```

## 段落写作

```text
请基于以下证据写一段综述正文。每个事实性论断后保留 [CitationKey]，不要编造文献。
```

## 审稿式检查

```text
请像审稿人一样检查这段综述：是否有逻辑跳跃、引用不足、过度声明、遗漏对立观点。输出修改建议。
```
